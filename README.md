# Deep Learning Lab Experiments

This repository contains all the practical experiments performed in the Deep Learning lab.  
The main purpose of this project is to understand important Deep Learning concepts through hands-on implementation using Python and Jupyter Notebook.

In this repository, different experiments are included to cover both basic and advanced Deep Learning topics such as vanishing gradient, overfitting, learning rate scheduling, CNN, transfer learning, sequence models like RNN/LSTM/GRU, LangChain, and model explainability using LIME.

Each experiment is implemented in a separate notebook or Python file for better understanding and easy execution.

## Experiments Included

This repository includes the following experiments:

- Experiment 1: Vanishing Gradient Problem
- Experiment 2: Overfitting in Deep Learning
- Experiment 3: Learning Rate Scheduling
- Experiment 4: Convolutional Neural Network (CNN)
- Experiment 5: Transfer Learning using Pre-trained CNN
- Experiment 6: RNN, LSTM, and GRU
- Experiment 7: Introduction to LangChain
- Experiment 8: Model Explainability using LIME

## Technologies Used

The following tools and technologies are used in this project:

- Python
- Jupyter Notebook
- TensorFlow / Keras
- Scikit-learn
- NumPy
- Pandas
- Matplotlib
- LIME
- LangChain

## How to Run the Project

To run this project on your system, first clone the repository and open it in your local environment.  
Install the required libraries and run the notebooks one by one using Jupyter Notebook.

Each notebook contains code, explanation, and output for the respective experiment.  
You can execute the cells step by step to understand the implementation and results.

## Learning Outcome

By performing these experiments, we can understand:

- Basic to advanced Deep Learning concepts
- How neural networks learn
- Problems like vanishing gradient and overfitting
- Optimization techniques like learning rate scheduling
- Image processing using CNN

- Model interpretability using LIME
